# ShiftTalkers

ShiftTalkers is a Twitter website clone for car enthusiasts to have a platform to socialize and connect with other fellow car enthusiasts. ShiftTalkers is a full stack project I created using a Flask backend and a React frontend during my enrollment as a student in App Academy's six month software engineering bootcamp. The goal of this project was to challenge myself in making a fullstack project that was different from my previous projects at App Academy and to better hone my skills as Software Engineer.
